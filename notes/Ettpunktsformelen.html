Ettpunktsformelen

Linje I:

  • Les av skjæring med y-aksen $\implies b = 1$
  • Gå en enhet til høyre, må da 0.5 opp for å treffe på linja $\implies a = \dfrac{1}{2}$
y=ax+by=12x+1‾‾\begin{aligned} y = ax + b \\ \underline{\underline{y = \dfrac{1}{2}x + 1}} \end{aligned}y=ax+by=21​x+1​​​

Linje II:

  • Skjæring med y-aksen $\implies b = 0$
  • Gå en enhet til høyre, må da 1 ned for å treffe på linja $\implies a = -1$
y=ax+by=−x‾‾\begin{aligned} y = ax + b \\ \underline{\underline{y = -x}} \end{aligned}y=ax+by=−x​​​

Ved regning

  • Bruk ettpunktsformelen (s. 18 i FS)
    • $\boxed{y - y_1 = a \cdot (x - x_1)}$
      • Skriv om: $\boxed{y = ax + b}$
      • a = stigningstall
      • (x, y) = vilkårlig punkt på linja
      • (x_1, y_1) = bestemt punkt på linja

❓ Ei linje går gjennom punktene (0 $\small(x_1)$, -2 $\small(y_1)$) og (2 $\small(x_2)$, 1 $\small(y_2)$). Bestem likninga for linja.

✏ Vi trenger ett punkt og stigningstallet. Finner a: $a = \dfrac{\Delta y}{\Delta x} = \dfrac{y_2 - y_1}{x_2 - x_1} = \dfrac{1 - (-2)}{2 - 0} = \underline{\dfrac{3}{2}}$ Velger punkt (0, -2) og setter inn i ettpunktsformelen: $y - y_1 = a (x - x_1)$ $y - (-2) = \dfrac{3}{2}(x - 0 )$ $y + 2 = \dfrac{3}{2}x$ $y = \dfrac{3}{2}x - 2$

❓ Ei linje er parallell med linja $y = - \dfrac{3}{4}x + 5$ og går gjennom punktet (1, 3). Bestem likninga for linja.

✏ Parallelle linjer $\implies$ samme stigningtall Altså er $a = - \dfrac{3}{4}$ og vi har punktet (1, 3): Ettpunktsformel: $y - y_1 = a(x - x_1)$ $y - 3 = - \dfrac{3}{4}(x - 1)$ $y = - \dfrac{3}{4}x + \dfrac{3}{4} + 3$ $y = - \dfrac{3}{4}x + \dfrac{3}{4} + \dfrac{12}{4}$ $\underbrace{y = - \frac{3}{4}x + \dfrac{15}{4}}_\text{\color{red}y = ax + b}$
