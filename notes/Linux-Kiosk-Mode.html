Linux kiosk mode

Denne prosedyren baserer seg på Ubuntu 20.04 LTS med Firefox/Chromium som app for kiosk-mode. I praksis kjører dette bare en instans av nettleseren i headless/koisk-mode over Gnome og fjerner så mange distraksjoner som mulig.

VNC-serveren kjører på port 5905 og kan kobles til med IP:PORT.

  1. Installer en LTS versjon av Ubuntu (testet på 20.04)
    1. Installer i minimal mode, ikke installer ekstra software + oppdateringer under oppsett
    2. Brukernavn: kiosk
  2. sudo apt-get update 
  3. sudo apt-get install -y vim openssh-server slick-greeter x11vnc 
    1. Bytt til lightdm når du får spørsmål om det
  4. Fjern distraksjoner fra system Settings >
    1. Enable Notifications → “Do Not Disturb” 
    2. Privacy → Screen Lock → “Blank Screen Delay”: “Never” 
    3. Disable Privacy → Screen Lock → “Automatic Screen Lock” & “Lock Screen on Suspend” & “Show Notification on Lock Screen”
    4. Disable Privacy → Diagnostics → “Send error reports to Canonical”: “Never”
    5. Sound → “System Volume”: 0
    6. __Sound → Input “Volume”: 0__
    7. __Power → Power Saving “Blank Screen”: “Never”__
    8. __Power → Suspend & Power Button “Automatic Suspend”: “Off”__

    9. sudo vim /etc/lightdm/lightdm.conf 

      [SeatDefaults] autologin-user=kiosk autologin-user-timeout=0 user-session=ubuntu greeter-session=unity-greeter

  5. sudo vim /etc/lightdm/lightdm.conf.d/50-myconfig.conf 

    [SeatDefaults] autologin-user=kiosk

  6. mkdir /home/kiosk/.config/autostart 

  7. vim /home/kiosk/.config/autostart/kiosk.desktop 

    [Desktop Entry] Type=Application Name=Kiosk Exec=/home/kiosk/kiosk.sh X-GNOME-Autostart-enabled=true

  8. vim /home/kiosk/kiosk.sh 

    #!/bin/bash export DISPLAY=:0

    Hide the mouse from the display

    unclutter &

    Start x11vnc

    /usr/bin/x11vnc -create -xkb -noxrecord -noxfixes -display :0 -auth /var/run/lightdm/root/:0 -rfbauth /home/kiosk/.x11vnc.pass -forever -rfbport 5905 &

    1. chmod +x /home/kiosk/kiosk.sh 
  9. For å bruke Firefox:
    1. sudo apt-get install -y firefox

    2. General > Startup | Make Default…
      Disable General > Language | Check your spelling as you type
      Disable General > Browsing | Enable picture-in-picture video controls
      Disable General > Browsing | Recommend extensions as you browse
      Disable General > Browsing | Recommend features as you browse
      Home > New Windows and Tabs | Homepage and new windows: Blank Page
      Home > New Windows and Tabs | New tabs: Blank Page
      Disable Privacy & Security > Logins and Passwords | Ask to save logins and passwords for websites
      Disable Privacy & Security > Logins and Passwords | Show alerts about passwords for breached websites
      Disable Privacy & Security > Firefox Data Collection and Use > Allow firefox to send technical and interaction data to Mozilla
      Disable Privacy & Security > Firefox Data Collection and Use > Allow firefox to make personalized extension recommendations
      Disable Privacy & Security > Firefox Data Collection and Use > Allow firefox to install and run studies

    3. I about:config  endre
      • gfx.webrender.all: true 
      • layers.acceleration.force-enabled: true 
      • browser.tabs.warnOnClose: false
      • browser.tabs.warnOnCloseOtherTabs: false
      • browser.send_pings: false 
      • beacon.enabled: false 
      • alerts.useSystemBackend: true 
      • extensions.pocket.enabled: false 
      • browser.sessionstore.resume_from_crash: false 
      • toolkit.startup.max_resumed_crashes: -1
      • browser.disableResetPrompt: true 

    4. Legg til dette nederst i /home/kiosk/kiosk.sh 

      Run Firefox in kiosk-mode

      /usr/bin/firefox –kiosk –private –new-window https://duck.com &

  10. For å bruke Chromium: 1. sudo apt-get install -y chromium-browser 2. Disable Autofill → Passwords → “Offer to save passwords” & “Auto Sign-in”  3. Disable Autofill → Payment methods → “Save and fill payment methods” & “Allow sites to check if you have payment methods saved”  4. Disable Autofill → Addresses and more → “Save and fill addresses” 5. Disable Advanced → Languages → “Spell check” & Language → “Offer to translate pages that aren’t in a language you read” 6. Gjør Chromium til default nettleser: Default Browser → “Make Default” 7. Legg til dette nederst i /home/kiosk/kiosk.sh 

    # Clear crash-flags if Chromium crashed to avoid warning banners sed -i ‘s/”exited_cleanly”:false/”exited_cleanly”:true/’ /home/kiosk/.config/chromium/Default/Preferences sed -i ‘s/”exit_type”:”Crashed”/”exit_type”:”Normal”/’ /home/kiosk/.config/chromium/Default/Preferences

    # Run Chromium in kiosk-mode /usr/bin/chromium-browser –kiosk https://duck.com &

  11. ⎵x11vnc -storepasswd passwordgoeshere /home/kiosk/.x11vnc.pass
  12. Fjern Gnome Keyring passord (forskjellig fra login passord) 1. Åpne “Passwords and Keys” 2. Høyreklikk på Passwords → Passwords | Login → “Change Password”  3. Skriv inn det eksisterende passordet og sett et nytt blankt passord
  13. reboot  1. Når maskinen får startet igjen skal Chromium være den eneste synlige applikasjonen 2. Om du trenger å gå tilbake til skrivebordet kan klikke windows-knappen eller drepe Chromium fra en terminal på en annen TTY (CTRL+ALT+F<1-12>)/ssh eller ALT+F4.

Auto-reload med cron

  1. `sudo apt-get install -y xdotool`
  2. echo $DISPLAY 
  3. crontab -e 
    1. 0 */2 * * * ( DISPLAY=:0 xdotool search --onlyvisible --class firefox/chromium windowactivate --sync \%@ key ctrl+F5 ) 

Auto update

Mulig du må installere pakken unattended-upgrades om den ikke er installert allerede

sudo apt-get install -y unattended-upgrades

sudo vim /etc/apt/apt.conf.d/50unattended-upgrades

Kommenter inn og endre disse linjene

”${distro_id}:${distro_codename}-updates”; Unattended-Upgrade::MinimalSteps “true”; Unattended-Upgrade::Remove-Unused-Kernel-Packages “true”; Unattended-Upgrade::Remove-Unused-Dependencies “true”; Unattended-Upgrade::Automatic-Reboot “true”; Unattended-Upgrade::Automatic-Reboot-Time “02:00”;

sudo vim /etc/apt/apt.conf.d/20auto-upgrades

Legg inn disse linjene

APT::Periodic::Update-Package-Lists “1”; APT::Periodic::Download-Upgradeable-Packages “1”; APT::Periodic::AutocleanInterval “7”; APT::Periodic::Unattended-Upgrade “1”;

Sjekk at det funker

sudo unattended-upgrades –dry-run –debug

eller

cat /var/log/unattended-upgrades/unattended-upgrades.log

Dim skjerm på tidsintervall

crontab -e

Dim skjermen kl. 17:00 hver dag, u-dim den kl. 07:00 hver hverdag

0 17 * * * ( DISPLAY=:0 xrandr –output “$(DISPLAY=:0 xrandr -q | grep ‘ connected’ | head -1 | awk ‘{print $1}’)” –brightness 0.2 ) 0 7 * * 1-5 ( DISPLAY=:0 xrandr –output “$(DISPLAY=:0 xrandr -q | grep ‘ connected’ | head -1 | awk ‘{print $1}’)” –brightness 1 )
